Kegiatan Workshop Web Development

Saya berkesempatan mengikuti Kegiatan Workshop Web Development dengan topik "Build Responsive Web using Tailwind dan Alpine.js" yang sangat bermanfaat. Dalam workshop ini, saya mempelajari bagaimana membangun website yang responsif dan interaktif menggunakan Tailwind CSS untuk styling yang cepat dan efisien serta Alpine.js untuk menambahkan fungsionalitas JavaScript yang ringan. Dengan bimbingan instruktur yang berpengalaman, saya dapat mempraktikkan langsung pembuatan berbagai komponen UI yang dinamis, seperti menu navigasi dan modals, tanpa memerlukan library JavaScript yang kompleks. Workshop ini memberikan wawasan baru dan keterampilan teknis yang sangat berguna dalam pengembangan web modern.

Pengalaman saya dalam mengikuti program S1 Sistem Informasi (SI) pada tahun 2024 sangatlah berkesan dan penuh dengan tantangan yang memperkaya pengetahuan serta keterampilan saya di bidang teknologi.
